# YourGP Study Centre — Planning Document

> Decisions confirmed with Jack on 2026-04-10. This file exists so Claude Code web has full context.

## Purpose

Build a web-based study centre for the YourGP team (~85 people) so they can level up and become leaders within the business. Each organisational document becomes a learning module with reading material, embedded questions, and progress tracking.

## Architecture Decisions

### Authentication
- **M365 / Entra ID SSO** — staff log in with existing YourGP Microsoft accounts
- Restricted to YourGP org tenant

### Database
- **Supabase** (Jack has an existing account)
- Stores: user progress, quiz scores, completion data, admin dashboard data

### Hosting
- **Not yet decided**
- Azure Static Web Apps ruled out (no Azure subscription)
- Vercel or Netlify suggested (free tier, M365 SSO works with either) — needs confirmation

## Content Format

### Source Material
6 markdown documents in this repo:

| File | Topic |
|------|-------|
| `Goals 2025.md` | Annual strategic goals, BHAG, KPAs |
| `Leadership at YourGP.md` | 15 leadership standards across 5 C's |
| `Organisational Chart March 2026.md` | Current org structure (48 GPs, 32 support staff, 5 business team) |
| `YourGP - Clarity (The Advantage).md` | 6 critical questions (Lencioni framework) |
| `YourGP - Values & Outcome Matrix.md` | Role progression model (Sailor → Mid-Shipman → Skipper → Commander) |
| `YourGP - Who We Are.md` | Vision, mission, values, flywheel |

### Study Experience
1. Each document broken into **smaller readable chunks**
2. Each chunk followed by **quiz questions** (embedded inline)
3. **Interactive quiz** experience per chunk
4. Questions **generated by Claude, reviewed and approved by Jack** before going live

## Progression Model

- **No minimum score** to complete a module — this is about exposure, not pass/fail
- **Unlimited retakes** allowed
- **Any order** — modules can be completed in any sequence
- **Admin dashboard** — leadership can see who's completed what and their scores
